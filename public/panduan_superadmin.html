MODUL PENYELIA (SUPERADMIN)

Panduan Onboarding Tenant

Panduan praktis bagi Penyelia / Superadmin dalam melakukan instalasi, pendaftaran, dan konfigurasi awal BUMDesa atau UMKM baru.

Menu Panduan

  • 1. Membuat Tenant
  • 2. Mendaftarkan User
  • 3. Inisialisasi CoA

Langkah A: Membuat Profil Tenant (BUMDesa / UMKM)

Proses inisialisasi entitas bisnis baru pada server SimpleAkunting.

1

Navigasi ke Menu Tenants Menu Utama

Masuk ke dashboard utama Superadmin, arahkan kursor ke sidebar kiri lalu klik menu dropdown Central Admin dan pilih menu Tenants.

2

Klik Tombol Tambah Tenant Aksi

Di pojok kanan atas halaman daftar Tenant, klik tombol hijau bertuliskan Tambah Tenant. Form isian modal akan muncul secara interaktif.

3

Isi Detail Bisnis Baru Form Isian

Lengkapi formulir dengan ketentuan data sebagai berikut:

  • Nama Bisnis: Nama resmi BUMDesa atau UMKM (contoh: BUMDesa Sampah Jaya).
  • Kode Unik: Kode unik subdomain tanpa spasi (contoh: sampahjaya). Kode ini akan menjadi URL akses mereka.
  • Tipe Bisnis: Pilih di antara tiga jenis tipe database sesuai jenis usaha:
    • Jasa (Tanpa kontrol stok barang)
    • Perdagangan (Dengan modul stok produk jadi)
    • Manufaktur (Dengan modul BoM, bahan baku, dan produksi)
Form Tambah Tenant Baru
Perhatian Penting!

Kode unik tenant harus ditulis menggunakan huruf kecil semua, tanpa spasi, dan tanpa karakter spesial untuk mencegah error pada konfigurasi domain web server.

Langkah B: Menambahkan User untuk Tenant Baru

Membuat kredensial akses pimpinan / pemilik tenant agar bisa mengelola akun akuntansinya sendiri.

1

Masuk ke Menu Users Global Menu Utama

Di bawah menu dropdown Central Admin pada sidebar kiri, klik menu Users Global.

2

Klik Tombol Tambah User Aksi

Klik tombol biru Tambah User Global yang terletak di pojok kanan atas halaman daftar user global.

3

Isi Formulir User Baru Form Isian

Isilah data kredensial direktur atau pemilik bisnis dengan panduan kolom berikut:

  • Nama Lengkap: Nama pemilik atau direktur penanggung jawab tenant.
  • Nama Pengguna (Username): Buat nama pengguna unik tanpa spasi (contoh: direktursampahjaya). Catatan: Username ini tidak boleh sama dengan username yang telah terdaftar sebelumnya di sistem global.
  • Kata Sandi (Password): Buat kata sandi awal yang mudah diingat oleh pengguna. Infokan kata sandi tersebut ke user bersangkutan, karena nantinya user dapat menggantinya sendiri secara mandiri via profil pengaturan akun mereka.
  • Hak Akses (Role): Pilih Admin (Tenant). Ini adalah role level tertinggi untuk entitas bisnis individu (mempunyai akses penuh ke data master, transaksi keuangan, dan laporan internal).
  • Jabatan: Tulis Direktur / Owner atau setara.
  • Penugasan Tenant: Cari dan pilih nama tenant yang baru saja Anda buat di Langkah A (contoh: BUMDesa Sampah Jaya). Ini akan mengunci basis data pengguna tersebut ke database tenant yang terisolasi secara aman.
Form Tambah User Global

Inisialisasi Otomatis Bagan Akun (Chart of Accounts)

Kabar gembira untuk kenyamanan administrasi Anda!

Pada sistem SimpleAkunting v3.5, Bagan Akun (Chart of Accounts / CoA) untuk tenant baru kini otomatis dibuat dan diinisialisasi secara otomatis saat Anda pertama kali menyimpan Tenant.

Template default yang sesuai dengan standar akuntansi dinas koperasi akan langsung dikloning ke database tenant baru. Hal ini memastikan user tenant dapat langsung mencatat transaksi kas, penjualan, atau pembelian pertamanya **tanpa perlu melakukan pembuatan akun kas/modal manual dari nol**.

Proses Siap Pakai!

User baru Anda dapat langsung mengunduh/membuat faktur dan mencatat pengeluaran kas begitu akun mereka selesai Anda daftarkan di menu Penyelia.
